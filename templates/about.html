About Palm Catch Game

🎯 How to Play

  • 1. Allow camera access when prompted
  • 2. Show your hand to make the spider web appear
  • 3. Move your hand to control the web position
  • 4. Catch falling balls to score points
  • 5. Avoid gray bombs with '!' symbol
  • 6. You have 3 lives - don't let balls escape!

🕹️ Game Features

Ball Types:
  • Red Balls: Normal balls (10 points)
  • Yellow Balls: Bonus balls (25 points)
  • Gray Balls: Bombs (lose 1 life)
Controls:
  • Hand Movement: Control web position
  • R Key: Restart game
  • Q Key: Quit game

🔧 Technology Stack

This game uses advanced computer vision technologies:

  • OpenCV: Real-time video processing
  • MediaPipe: Advanced hand tracking (if available)
  • Flask: Web framework for the application
  • Bootstrap 5: Responsive web design
